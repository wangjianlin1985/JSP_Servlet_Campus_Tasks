# JSP_Servlet_Campus_Tasks
基于jsp+servlet+mysql校园任务管理系统毕业源码案例设计

## 功能实现：
  管理员角色包含以下功能：管理员登陆,用户管理,任务管理,发布任务,管理员审核任务等功能。
  用户角色包含以下功能：用户登录,查看任务广场,操作任务,个人信息管理,消息提示等功能。
  用了技术框架： HTML+CSS+JavaScript+jsp+servlet+dao+mysql
